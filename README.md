# TP - Gestión Colaborativa, Control de Versiones y Organización Empresarial

## 📌 Descripción del Proyecto

Este proyecto fue desarrollado para el Trabajo Práctico de la materia Organización Empresarial de la Tecnicatura Universitaria en Programación (UTN).

El objetivo principal es aplicar herramientas de trabajo colaborativo y control de versiones utilizando Git, GitHub y Jira, junto con el desarrollo de un análisis básico de datos en Python.

El escenario elegido corresponde al:

## 📊 Escenario B - Análisis de Ventas de una Pequeña Empresa

Se realizó un análisis simple sobre un conjunto de datos de ventas comerciales simuladas correspondientes al año 2024.
